# map_app
Introduction: 
It is a Flutter project which is an Android mobile app (alse works on web-Chrome) featuring a Google Map using the Flutter SDK.


# Codelab
https://codelabs.developers.google.com/codelabs/google-maps-in-flutter/

# Notification
Please input your own Google Map API keys for accessing the map feature.
For Android: edit the "AndroidManifest.xml" file in "android/app/src/main".
For Web: edit the "index.html" file in "web".

You can create and find your keys from https://console.cloud.google.com/google/maps-apis/credentials
